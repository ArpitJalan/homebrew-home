# Homebrew @ $HOME

Homebrew install for those that like to Homebrew @ $HOME (i.e. ~/.homebrew).


## Install

    % bash < <(curl -s https://raw.github.com/wilmoore/homebrew-home/master/go)


## Un-Install

    % rm -rf ~/.homebrew
